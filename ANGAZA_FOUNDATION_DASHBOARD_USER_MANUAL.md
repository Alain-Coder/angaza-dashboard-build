## Introduction

Welcome to the Angaza Foundation Dashboard User Manual. This comprehensive guide will help you navigate and utilize all features of the dashboard system designed to empower and uplift marginalized communities in Malawi.

The Angaza Foundation Dashboard is a web-based application that provides tools for managing foundation operations, including project tracking, financial management, resource distribution, personnel management, and community impact measurement.

## System Requirements

### Browser Requirements

The dashboard is compatible with the following web browsers:
- Google Chrome (latest version)
- Mozilla Firefox (latest version)
- Microsoft Edge (latest version)
- Safari (latest version)

### Device Requirements

- Desktop or laptop computer
- Tablet or smartphone with modern web browser
- Stable internet connection

### Screen Resolution

For optimal viewing experience:
- Minimum resolution: 1024x768
- Recommended resolution: 1366x768 or higher

## Getting Started

### Account Setup

Before accessing the dashboard, you must have an account created by the System Administrator. The System Administrator will:

1. Create your user account in the system
2. Assign you to a specific role
3. Provide your login credentials

If you don't have an account, contact the System Administrator at admin@angazafoundation.org.

### Login Process

1. Navigate to the dashboard URL provided by your administrator
2. Enter your email address in the email field
3. Enter your password in the password field
4. Click the "Sign In" button
5. You will be automatically redirected to your role-specific dashboard

## Core Modules

### Overview Dashboard

The Overview Dashboard is your central hub for monitoring foundation activities.

**Accessing the Dashboard:**
1. Log in to the system
2. You will be automatically directed to your role-specific dashboard

**Dashboard Components:**
- **Key Metrics**: Displays important statistics based on your role
- **Recent Activity**: Shows recent system activities
- **Quick Actions**: Provides shortcuts to common tasks

**Using the Dashboard:**
- View real-time metrics relevant to your role
- Click on quick action buttons to navigate to specific modules
- Monitor recent activities to stay updated on system changes

### Project Management

The Project Management module allows you to create, track, and manage foundation projects.

**Creating a New Project:**
1. Navigate to the "Projects" section in the sidebar
2. Click the "Add Project" button
3. Fill in project details:
   - Project name
   - Description
   - Start and end dates
   - Budget allocation
   - Assigned team members
4. Click "Save Project"

**Managing Existing Projects:**
- View project details by clicking on a project in the list
- Edit project information using the "Edit" button
- Track project progress through status updates
- Monitor budget spending and resource allocation

**Project Status Options:**
- Planning
- Active
- Completed
- On Hold

### Personnel Management

The Personnel Management module handles staff and volunteer information.

**Attendance Tracking:**
Eligible roles (Finance Lead, Programs Lead, Project Officer, Office Assistant, Admin Officer, Community Outreach Officer) can track their work hours:

1. **Clocking In:**
   - Click the "Clock In" button in the sidebar
   - Select a project to associate with your time entry
   - Click "Start Working"

2. **Taking Breaks:**
   - Click the "Break" or "Lunch" button while clocked in
   - Click "End Break" when returning to work

3. **Clocking Out:**
   - Click the "Clock Out" button when finishing work
   - System automatically calculates total time and overtime

**Timesheet Management:**
- View your timesheet history in the Staff section
- Executive Directors can approve/reject timesheets
- Board Members can view but not approve timesheets

### Donation Management

The Donation Management module tracks financial contributions to the foundation.

**Viewing Donations:**
1. Navigate to the "Donations" section in the sidebar
2. View all donations in the table format
3. Filter donations by date, donor, or project

**Donation Information:**
- Donor name and contact information
- Donation amount and currency
- Payment method
- Associated project (if applicable)
- Date of donation

### Financial Management

The Financial Management module handles fund requests and liquidations.

**Creating a Fund Request:**
1. Navigate to the "Finance" section
2. Click "New Request" button
3. Fill in request details:
   - Amount requested
   - Purpose of request
   - Department and project (if applicable)
4. Submit request for approval

**Liquidation Process:**
1. After a request is approved, you can create a liquidation
2. Click "Liquidate" next to the approved request
3. Add receipts for expenses:
   - Upload receipt files
   - Enter expense details
   - Categorize expenses
4. Submit liquidation for approval

**Financial Status Tracking:**
- Pending: Awaiting approval
- Approved: Request approved, awaiting liquidation
- Rejected: Request denied with reason
- Submitted: Liquidation submitted for approval
- Completed: Liquidation approved

### Resource Distribution

The Resource Distribution module manages inventory and distribution of supplies.

**Managing Inventory:**
1. Navigate to "Resources" in the sidebar
2. View current inventory in the "Inventory" tab
3. Add new resources using the "Add Resource" button
4. Edit existing resources using the edit icon
5. Delete resources using the trash icon

**Creating a Distribution:**
1. In the "Inventory" tab, click "New Distribution"
2. Select a resource from available inventory
3. Enter quantity to distribute
4. Specify recipient and location
5. Add any notes about the distribution
6. Click "Distribute Resource"

**Stock Monitoring:**
- Green indicator: Adequate stock levels
- Yellow indicator: Low stock (less than 10 units)
- Red indicator: Out of stock (0 units)

### Beneficiary Management

The Beneficiary Management module tracks individuals and communities served by the foundation.

**Adding a Beneficiary:**
1. Navigate to the "Beneficiaries" section
2. Click "Add Beneficiary" button
3. Enter beneficiary details:
   - Full name
   - Age and gender
   - Community/location
   - Contact information (if available)
   - Program enrollment
4. Save beneficiary information

**Beneficiary Information:**
- Personal demographics
- Program participation
- Service delivery history
- Impact assessment data

### Program Portfolio

The Program Portfolio module manages the foundation's various programs.

**Creating a Program:**
1. Navigate to "Program Portfolio" section
2. Click "Add Program" button
3. Enter program details:
   - Program name and description
   - Associated project
   - Budget allocation
   - Timeline
   - Assigned staff members
4. Save program information

**Program Management:**

- Monitor budget utilization

- Measure program impact

### Partners Management

The Partners Management module maintains relationships with collaborating organizations.

**Adding a Partner:**
1. Navigate to the "Partners" section
2. Click "Add Partner" button
3. Enter partner details:
   - Organization name
   - Contact person and information
   - Partnership type
   - Active projects
4. Save partner information

**Partner Information:**
- Organization profile
- Contact details
- Collaboration history
- Active projects and involvement

### File Management

The File Management module allows users to store and organize documents.

**Uploading Files:**
1. Navigate to the "Files" section
2. Click "Upload File" button
3. Select a file from your device
4. Choose a folder or create a new one
5. Click "Upload"

**Creating Folders:**
1. In the Files section, click "New Folder"
2. Enter a folder name
3. Click "Create"

**Managing Files:**
- Rename files or folders using the three-dot menu
- Delete files or folders using the three-dot menu
- Preview files by clicking the eye icon
- Download files by clicking the download icon

**Navigation:**
- Use breadcrumbs to navigate folder hierarchy
- Switch between grid and list views
- Search for files using the search bar

## Attendance and Timesheet Management

Staff members with eligible roles can track their work hours using the attendance system.

### Clocking In and Out

1. **Clock In:**
   - Click the "Clock In" button in the sidebar
   - Select a project from the dropdown (or "General Work")
   - Click "Start Working"

2. **Taking Breaks:**
   - While clocked in, select "Regular Break" or "Lunch Break" from the dropdown
   - Click the appropriate button to start the break
   - Click "In Office" when returning from break

3. **Clock Out:**
   - Click the "Clock Out" button when finishing work
   - System automatically calculates:
     - Total time worked
     - Overtime hours (after 4:30 PM)
     - Break time

### Timesheet Approval

Executive Directors can approve or reject staff timesheets:

1. Navigate to the "Staff" section
2. Click on the "Timesheets" tab
3. Review pending timesheet requests
4. Click the checkmark to approve or X to reject
5. Add comments if rejecting a timesheet

## Troubleshooting

### Common Issues and Solutions

**Login Problems:**
- Ensure you're using the correct email and password
- Check that Caps Lock is off
- Clear your browser cache and cookies
- Try using a different browser

**Loading Issues:**
- Check your internet connection
- Refresh the page
- Clear browser cache
- Try again later if the system is busy

**File Upload Problems:**
- Ensure file size is less than 5MB
- Check that the file type is supported
- Verify you have a stable internet connection

**Navigation Issues:**
- Use the breadcrumb trail to ensure you're in the correct location
- If navigation isn't working, try refreshing the page

### Error Messages

**"Permission Denied":**
- You don't have access to this feature
- Contact your System Administrator if you believe this is an error

**"Data Not Found":**
- The requested information doesn't exist
- Check that you're looking in the correct section

**"Network Error":**
- Connection to the server was lost
- Check your internet connection and try again

## FAQ

**Q: How do I reset my password?**
A: Contact the System Administrator to reset your password.

**Q: Can I access the dashboard from my mobile device?**
A: Yes, the dashboard is responsive and works on tablets and smartphones.

**Q: What should I do if I forget to clock out?**
A: Contact your supervisor or the System Administrator to adjust your timesheet.

**Q: How often is my data backed up?**
A: Data is automatically backed up through Firebase's built-in backup systems.

**Q: Can I edit information after it's been submitted?**
A: Some information can be edited, while other information is locked after submission. Contact your supervisor for assistance.
